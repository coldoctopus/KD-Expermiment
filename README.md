# KD-Expermiment
This repo is an experiment of Konwledge Distillation for reasoning. based on a paper, we will replicate the process in this repo.

## Setup using Python venv

### Prerequisites
- Python 3.10+ installed and available on PATH
- Verify: `python --version`

### Create a virtual environment
- `python -m venv .venv`

### Activate the environment
- macOS/Linux: `source .venv/bin/activate`
- Windows (PowerShell): `.\.venv\Scripts\Activate.ps1`
- Windows (cmd): `.\.venv\Scripts\activate.bat`

### Install dependencies
- `pip install --upgrade pip`
- `pip install -r requirements.txt`

### Run the notebook
- `jupyter notebook`
- Open `experiment.ipynb`

### VS Code tip
- Select the `.venv` interpreter in the Command Palette (Python: Select Interpreter) before running notebooks.
